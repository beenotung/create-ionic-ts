# create-ionic-ts

Minimal [Ionic](https://ionicframework.com/) + [Capacitor](https://capacitorjs.com/) starter template: plain TypeScript and `<ion-*>` in HTML — no frontend framework.

[![npm Package Version](https://img.shields.io/npm/v/create-ionic-ts)](https://www.npmjs.com/package/create-ionic-ts)

## Usage

1. Create a new project folder

```bash
npm init ionic-ts [your-project-folder]
```

If the folder path is not given, it will be asked interactively.

2. Install dependencies and run the project

Installation is fully controlled by you, so you can use any package manager you like.

```bash
cd <your-project-folder>
npm install   # or: pnpm install, yarn
```

Use `npm start` to start the development server with automatic rebuilds when sources change.
